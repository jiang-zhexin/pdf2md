# PDF To Markdown

Extract text and structured content from PDF documents.

[![Deploy on Deno](https://deno.com/button)](https://console.deno.com/new?clone=https://github.com/jiang-zhexin/pdf2md)

> [!IMPORTANT]
> Add `MISTRAL_API_KEY` environment variable.

## Quickstart

Go to [demo page](https://pdf2md.zhexin.org).
